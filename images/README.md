## 📖 Overview

The **AI-Powered Document Intelligence System** is an intelligent document analysis application developed using **Python**, **Streamlit**, and **Google Gemini AI**. It enables users to upload **PDFs, Excel files, and Images** and interact with them through a conversational AI interface.

The system extracts text from documents, performs OCR on scanned PDFs, analyzes images, summarizes content, extracts key entities, and answers user questions using Google's Gemini model.

This project was developed as part of the **CPCL Internship** to demonstrate the practical use of **Generative AI**, **Document Intelligence**, and **Natural Language Processing (NLP)**.

# ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Powered-Document-Intelligence-System.git
```

### Navigate to the Project

```bash
cd AI-Powered-Document-Intelligence-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Gemini API Key

Replace the API key inside `text.py` with your own Google Gemini API Key.

### Run the Application

```bash
streamlit run login.py
```

# 🚀 How It Works

1. Login to the application.
2. Upload a PDF, Excel file, or Image.
3. The system extracts document content.
4. OCR processes scanned documents.
5. Gemini AI analyzes the extracted information.
6. Users can ask questions in natural language.
7. The chatbot returns accurate AI-generated responses.
